Partner Contact Nutrition Diet
==============================

This module adds the nutrition diet to a partner contact:

* Lifestyle
* Carnivore
* Pescatarian
* Vegetarian

Usage
-----






=====

* Go to Contacts
* Create or select a contact
* Go to the Nutrition tab to select their diet
